# Godot4 Universal Game Framework Docs

This directory houses the technical documentation for the Godot4 Universal Game Framework.

- [`architecture.md`](./architecture.md) – high-level architectural overview and data flow diagrams.
- [`modules/`](./modules) – detailed guides for each kit, backend adapter, and support component.
- [`modules/backend_functions.md`](./modules/backend_functions.md) – cloud function templates for Supabase/Firebase deployments.
- [`modules/supabase_schema.sql`](./modules/supabase_schema.sql) – reference database schema for Supabase projects.

Created as part of the modular framework initiative to support Supabase and Firebase backends.
