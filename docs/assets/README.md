# 预览素材采集指南

为了便于在 Markdown 文档中预览每个效果，请在录制或截屏后将媒体资源保存到本目录，并遵循以下约定：

1. **命名规则**：`<effect_name>.<ext>`，例如 `explosion.gif`、`laser_beam.png`。推荐使用 GIF，若大小受限可使用 PNG。
2. **分辨率**：保持 720p（1280×720）或更低，避免超过 5 MB。
3. **采集方法**：
   - 在 Godot 编辑器或 Demo 场景中按 `F8` 运行目标场景。
   - 使用屏幕录制工具录下 2 秒循环片段，或截取最具表现力的帧。
   - 如需透明背景，可在 2D 场景中启用 `CanvasItem` 的透明背景并在后期裁剪。
4. **Markdown 引用**：在效果页面使用相对路径，例如：
   ```markdown
   ![Explosion 演示](../assets/explosion.gif)
   ```
5. **版权说明**：素材沿用仓库原始许可（代码 MIT，素材 CC-BY 4.0），引用时请保留出处。
